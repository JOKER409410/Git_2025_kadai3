# Git_2025_kadai3

映画未選択時のレコメンデーション手法と理由
手法：平均評価ベースのレコメンデーション
理由:全体的に評価の高い映画を提示することで、初回利用時でも妥当な推薦が可能だから

映画選択時のレコメンデーション手法と理由
手法：ユーザーベース協調フィルタリング
理由:同じ映画を好むユーザーの行動を基に、嗜好に近い映画を推薦できるため

処理内容：
1.選択映画を評価しているユーザーを抽出
2.そのユーザーが評価した「別の映画」を加点対象とする
3.評価値をスコアとして累積加算
4.最終的にスコアが高い映画を上位5件抽出

Webアプリケーションの起動手順
① 必要な環境
Python 3.x

② 必要なライブラリのインストール
pip install flask

③ アプリケーションの起動
python app.py

④ ブラウザでアクセス
http://127.0.0.1:5000
